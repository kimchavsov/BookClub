# BookClub

BookClub is an application that use for tracking book progress and note taking.

## Getting Started

First, install all the dependencies:

```bash
npm install
```

To start the application:

```bash
node app.js
```

## Languages and Tools:

Languages and Framework:
- HTML & CSS
- EJS
- JavaScript
- Express
- Passport.js for auth

Tools:
- MongoDB
- NodeJS
